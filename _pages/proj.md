I've been fortunate to work on some really cool projects over the years. Below are some that have received media coverage.

Allocating Police Officers using 911 Data in Chicago
=====

In 2019, the Chicago Police Department and the Mayor's Office asked the University of Chicagod to conduct a staffing analysis and build a model that could be used to allocate police officers across the city. To solve this problem, we analyzed historical 911 data and determined which police units responded to the most 911 calls in prior recent periods. The model would then allocate more police resources towards these busy units.

This project was highly political, and has been covered several times by Chicago newspapers.
